Eagle

EAGLE (Enhanced Agents for Generative Language Excellence) is a framework for developing multiagent systems based upon cognitive mechanisms. It was inspired by the conceptual framework presented in Cognitive Architectures for Language Agents (CoALA). EAGLE is currently being developed in Python 3 on top of LangGraph.

Multiagent Framework Evaluation

CELMAS (Complete Evaluation for LLM Multi-Agent Systems) is a complete evaluation framework for LLM multi-agent systems. We are investigating and developing metrics and methods to evaluate collaborative LLM agents. This framework defines and implements techniques for assessing and analyzing LLM agents’ behavior. The goal is to assist with, facilitate, and document the testing of the multi-agent system in a systematic and reproducible manner.

Oil & Gas Glossary

Development of a glossary of terms and acronyms, including detailed descriptions, covering the operational routines of FPSO (Floating Production Storage and Offloading) units. The glossary also incorporates definitions of key tags (unique identifiers assigned to instruments, equipment, valves). This resource has been systematically reviewed and expanded by a domain expert in the oil and gas industry, ensuring technical accuracy and consistency.
