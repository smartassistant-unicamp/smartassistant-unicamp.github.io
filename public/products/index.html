Eagle

EAGLE (Enhanced Agents for Generative Language Excellence) is a comprehensive framework for developing multi-agent systems grounded in cognitive mechanisms. EAGLE is currently being developed in Python 3 on top of LangGraph.

Multiagent Evaluation Framework

CELMAS (Complete Evaluation for LLM Multi-Agent Systems) is a complete evaluation framework for LLM multi-agent systems. We are investigating and developing metrics and methods to evaluate collaborative LLM agents using automatically generated test cases. This framework defines and implements techniques for assessing and analyzing the behavior of LLM agents. The goal is to assist with, facilitate, and document the testing of the multi-agent system in a systematic and reproducible manner.

Oil & Gas Glossary

Development of a glossary of terms and acronyms, including detailed descriptions, covering the operational routines of FPSO (Floating Production Storage and Offloading) units. The glossary incorporates definitions of key tags (unique identifiers assigned to instruments, equipment, and valves). This resource has been systematically reviewed and expanded by a domain expert in the oil and gas industry, ensuring technical accuracy and consistency.
