{{<image-text-row img="/img/products/multiagent-framework.png" title="Multiagent Evaluation Framework">}}
CELMAS (**C**omplete **E**valuation for **L**LM **M**ulti-**A**gent **S**ystems) is a complete evaluation framework for LLM multi-agent systems. We are investigating and developing metrics and methods to evaluate collaborative LLM agents using automatically generated test cases. This framework defines and implements techniques for assessing and analyzing the behavior of LLM agents. The goal is to assist with, facilitate, and document the testing of the multi-agent system in a systematic and reproducible manner.
